**Lab1_FX: Конвертер на базе MVC**

Данный проект является выполнением первой лабораторной работы по разработке графического приложения на языке Java.
Основная цель проекта — освоение библиотеки JavaFX и реализация архитектурного паттерна Model-View-Controller (MVC).

**Функциональные возможности**

Приложение представляет собой конвертер величин и поддерживает следующие возможности:

	•	Поддержка как минимум трёх единиц измерения
	•	Работа с дробными числами
	•	Обработка ошибок ввода и валидация данных

**Архитектура проекта**

Программа разделена на три основных компонента, что обеспечивает чистоту и удобство поддержки кода:

	1.	Model (ConverterModel): Содержит математическую логику вычислений и коэффициенты конвертации. Не зависит от пользовательского интерфейса.
	2.	View (ConverterView.fxml): Определяет структуру графического интерфейса, включая: поля ввода (TextField), кнопки (Button), метки для вывода результата (Label)
	3.	Controller (ConverterController): Управляет взаимодействием пользователя с программой, связывая модель и представление.

**Технологический стек**
	•	Язык программирования: Java
	•	Графический интерфейс: JavaFX
	•	Система сборки: Maven
